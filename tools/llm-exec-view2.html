3. Attention 内部机制

Q/K/V 核心运算流

1. 线性投影:
Q = X · W_q + RoPE
K = X · W_k + RoPE
V = X · W_v
2. 注意力分数:
S = (Q · K^T) / sqrt(d) + Mask
3. 归一化求和:
Output = Softmax(S) · V

8. 优化全景核心科技

📖 PagedAttention

类似虚拟内存机制，非连续存储 KV Cache，极大缓解显存碎片化。

⚡ Flash Attention

Tiling 切块技术，将计算保留在极速 SRAM 中，减少 HBM 读写。

🔄 Continuous Batching

动态批处理，请求来一条算一条，退一条补一条。

📉 Quantization

将 FP16 权重压缩为 INT8/INT4，本地部署大模型的基石。

🔮 投机解码

小模型快编候选词，大模型并行验证，打破逐字生成瓶颈。

🧠 Prompt Caching

复用前缀（System Prompt），实现首字极速响应。
