一次请求的关键时间线

~5ms网络接入

客户端请求到 API 网关。

~1ms调度排队

服务系统决定 batch 和 GPU。

~0.5msTokenize

文本切成 token id。

50-200msPrefill

整段 prompt 并行过模型。

TTFT首 token

首个 logits 采样后返回。

20-40ms/tokDecode

每步读取历史 KV Cache。
